# MVP boundaries (Phase 1)

## Allowed

- Official site discovery  
- Bounded **same-domain** crawl  
- Press / news parsing  
- Careers parsing  
- GitHub collector (public API, token via env)  
- Rules-based scoring  
- CSV / Markdown reports  

## Not allowed in Phase 1 unless explicitly approved

- Playwright  
- Scrapy  
- LinkedIn scraping  
- LLM-based scoring  
- GUI implementation  
- Chatbot implementation  
- Graph implementation (including persisting entity/edge graphs)  
- OCR  
- Broad third-party review / forum crawling  

Keep **crawl depth**, **page count**, and **request rate** conservative.
