# Project scope — prj_InVendX

- This project is **prj_InVendX**.
- **Purpose**: evidence-first vendor intelligence pipeline for wealthtech / fintech vendor analysis.
- **Phase 1** uses **public sources only**.
- This is **not** a general-purpose web scraper; scope stays vendor research and evidence extraction.
- This is **not** a chatbot project yet.
- This is **not** a GUI project yet.
- This is **not** a graph database project yet.
- Optimize for **modular Python**, **SQLite-centered** storage, and **traceable evidence rows**.
- **All scores and reports** must trace back to **evidence rows** and **source URLs**.
